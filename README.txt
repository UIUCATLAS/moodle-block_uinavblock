The uinavblock block adds a block to your course that allows students to quickly 
jump to any section in the course.  The display of the navigation list can be changed 
use the names and/or numbers of the sections and can be displayed either horizontally
or vertically.  The block can be placed in any region of the page, as well as a speical 
setting to place the block in the main content region even if that region is not defined in the theme   

--------------------------------------------------------------------------------
To Install:

1. Load the uinavblock directory into your "blocks" subdirectory.
2. Visit your admin page to create complete installation.
